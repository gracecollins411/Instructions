How to Find

  • First you need to find a reliable and affordable 3D printer. I like the website Creality 3D and the Ender 3 Pro is a good deal.

How to Use

  • You will get the printer in parts so watch this video:
  • After you assemble the 3D printer choose an STL file from any resource you want. A good website is Thingiverse:
  • Once you find a file that you are happy with download it. A zip file should then appear and double click it to access the STL.
  • You will then have to transfer the STL file to a software that will be able to connect to your 3D printer. I like the software Cura by Ultimaker:
  • Upon dragging the STL file to Cura you will be able to resize and scale the 3D print.

How to Print

  • Plug the USB with file card that came with the printer into your computer.
  • Save the file to the USB.
  • Push the card into the 3D printer.
  • Turn on the 3D printer
  • Click on "print from TF" and select your file.

  • Now the plate will start heating up and your print will be on its way!
